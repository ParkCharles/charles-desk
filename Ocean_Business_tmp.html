데이터 기반 항해 성과·탄소 관리 서비스

1. 왜 지금인가 (Why Now)

  • 전 세계 해운업은 연료비·탄소배출·ESG 규제로 인해 “운항 성과를 수치로 설명하고, 개선해야 하는 시대”에 진입
  • 대한민국은 2026년 해양수산부 부산 이전 첫해 → 해양 스타트업 지원, 예산, 정책 실험이 집중될 가능성 높음
  • 하지만 현실은:
    • 항해 데이터는 흩어져 있고
    • 보고서는 사후 정리용에 그치며
    • “다음 항해에서 무엇을 바꿔야 하는지”를 명확히 알려주는 도구는 거의 없음

👉 문제는 명확하지만, IT로 제대로 풀린 적이 없음

2. 우리가 푸는 문제 (Problem)

실제 현장의 문제

  • 항해 후 보고서는 있지만 복기와 개선이 없음
  • 연료비·지연·탄소 배출의 책임은 개인에게 귀속되지만 → 성과를 정량적으로 설명할 도구가 부족
  • ESG·탄소 보고는 시간이 많이 들고, 경영진 설득도 어려움

핵심 질문

“이 항해에서 우리는 돈을 얼마나 더 썼고,
다음 항해에서는 무엇을 바꾸면 줄일 수 있는가?”

3. 명확한 고객 (Single Buyer Persona)

중견 해운사의 운항/용선/선박 관리 책임자 (Fleet Manager / Technical Superintendent)

이 사람은:

  • 항해 성과에 직접 책임을 지고
  • 연료비·탄소·지연을 줄이면 바로 자기 성과
  • 분기/연말 보고서를 위에 올려야 하는 실무자
  • 작은 개선(1~3%)도 즉시 돈이 되는 위치

👉 처음부터 선주/경영진이 아니라
👉 “성과 압박을 받는 실무 책임자”가 첫 고객

4. 우리가 만드는 해결책 (Solution)

핵심 한 문장

“지난 항해를 데이터로 복기해서
다음 항해에서 돈과 리스크를 줄여주는 도구”

제공 가치

  • 항해 데이터를 기반으로:
    • 연료 효율
    • 지연 요인
    • 탄소 배출
  • Before / After 시뮬레이션
    • “이렇게 했으면 % 개선 가능”
  • 결과는:
    • 대시보드
    • 분기/반기/연말 보고서(PDF)

👉 데이터 그 자체가 아니라 의사결정용 요약이 핵심

5. MVP 전략 (2인 창업 현실 기준)

Phase 1 — 즉시 돈 되는 모델

반자동 분석 + 리포트 제공 (컨설팅 × SaaS)

  • 입력:
    • 공개 가능한 과거 항해 데이터
    • IMO DCS / EU MRV / 연차보고서
  • 출력:
    • 항해별 성과 분석
    • 잠재적 비용 절감 수치
    • 탄소 배출 및 CII 영향
  • 가격:
    • 건당 / 월 단위 과금
    • 초기엔 수작업 + 자동화 혼합

👉 실데이터 연동 없이도 시작 가능
👉 바로 매출 발생

Phase 2 — 반복 가능한 SaaS

  • 항해 데이터 업로드
  • 자동 분석
  • 보고서 자동 생성
  • ESG / 탄소 대응 포맷 제공

Phase 3 — 해자(Moat)

  • 선박 유형별 벤치마크
  • 업계 평균 대비 위치
  • 데이터 네트워크 효과

6. 탄소·ESG를 “돈의 언어”로 번역

  • “탄소를 줄이자” ❌
  • “이 항해의 탄소 X톤 = 비용 ○○원” ✅
  • “CII 등급 리스크 감소” ✅

👉 ESG는 명분,
👉 실제 고객이 사는 건 비용 절감과 리스크 관리

7. 역할 분배

해양 전문가

  • 실제 용선·운항 계약 경험
  • 현장의 언어와 신뢰
  • 어떤 데이터가 의미 있는지 알고 있음

IT 창업가

  • AI·데이터 분석
  • 자동화된 보고·대시보드
  • 서비스·SaaS 구조화

👉 이 두 가지가 동시에 있어야만 가능한 사업

8. 얻을 수 있는 것

  • 단순 중개나 컨설팅이 아닌 지속 가능한 제품 회사
  • 부산을 베이스로 한 해양 데이터 SaaS
  • 향후:
    • 선사
    • 금융
    • 보험
    • 탄소 시장 확장 가능성

9. 첫 실행 계획 (1개월)

  1. 실제 항해 사례 1~2건 선정
  2. 공개 데이터 기반 분석 적용
  3. 가짜 대시보드 + PDF 리포트 제작
  4. 기존 네트워크에 무료 1회 분석 제안
  5. 반응 있는 고객부터 유료 전환

10. 진짜 가치

“다음 항해에서,
당신이 덜 욕먹고,
덜 손해 보고,
성과를 설명할 수 있게 해줍니다.”
