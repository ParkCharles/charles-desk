# 2025 비즈니스 AI 현황 보고서

MIT Project NANDA와 협력하여 제작된 기업의 생성형 AI 도입 현황에 대한 종합 보고서입니다.

## 📊 보고서 개요

이 보고서는 2025년 현재 기업들이 생성형 AI를 어떻게 도입하고 있는지, 그리고 성공과 실패를 가르는 요인이 무엇인지 분석합니다.

### 주요 주제

- **생성형 AI 간극(Generative AI Gap)**: 파일럿과 프로덕션 배포 사이의 간극
- **산업별 AI 혁신**: 다양한 산업의 AI 성숙도와 전략
- **구축 vs 구매**: 기업의 AI 전략 변화
- **에이전틱 시스템**: 자율 학습하는 차세대 AI
- **고용 영향**: AI가 인력에 미치는 실제 영향

## 🚀 GitHub Pages 설정 방법

1. 이 저장소를 GitHub에 푸시
2. Settings → Pages로 이동
3. Source를 "Deploy from a branch" 선택
4. Branch를 "main" 또는 "master" 선택, 폴더는 "/ (root)" 선택
5. Save 클릭
6. 몇 분 후 `https://[username].github.io/[repo-name]` 에서 확인 가능
